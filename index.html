Sobre

O AnimeFire é um dos sites mais antigos de animes do Brasil, com um vasto catálogo. O AnimeFire Downloader é uma ferramenta criada para facilitar o download de episódios diretamente do site AnimeFire.

Funcionalidades

  • Extrai automaticamente os links de episódios de uma página principal do AnimeFire.
  • Permite escolher a qualidade desejada para os downloads (SD, HD, F-HD).
  • Opção de baixar todas as qualidades disponíveis ou apenas a qualidade preferida.
  • Organiza os vídeos baixados em pastas com o nome do anime e número do episódio.

Como Usar

Clone o repositório, instale as dependências e execute o script Python para começar a baixar seus episódios favoritos de anime.

git clone https://github.com/e43b/AnimeFire-Downloader.git
cd AnimeFire-Downloader/codes
python main.py

Exemplo de Uso

Insira o link da página principal do anime para começar a baixar os episódios.

Insira o link da página principal do anime: https://animefire.plus/animes/one-piece-dublado-todos-os-episodios
